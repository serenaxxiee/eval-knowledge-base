# Eval Knowledge Base — Claude Code Instructions

## What this repo is
A git-versioned knowledge base of PUBLIC AI agent evaluation content. Updated daily by automated research.

## How to search
- Browse by folder: `microsoft/`, `industry/`, `concepts/`
- Each file has YAML frontmatter with `title`, `tags`, `last_updated`
- Use grep to find content by keyword across all files
- Check `changelog/` for the most recent additions

## How to answer questions from this repo
1. Search for relevant files by topic/keyword
2. Read the TL;DR and Key Facts for a quick answer
3. Read the Detail section if more depth is needed
4. Always cite the Sources when referencing specific claims

## How to add content
- One file per topic, in the appropriate folder
- Use the layered format: frontmatter → TL;DR → Key Facts → Customer Impact → Detail → Sources
- Update existing files rather than creating duplicates
- Add a changelog entry for each day's changes

## Content routing
This is the PUBLIC repo. Only commit content that is:
- From publicly available sources (Microsoft Learn, GitHub, blogs, papers)
- Not competitive analysis or internal Microsoft content
- Not product roadmap signals or internal tool details
