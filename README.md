# Eval Knowledge Base

A comprehensive, continuously-updated knowledge base for AI agent evaluation — covering frameworks, tools, research, and best practices.

## What's here

- **microsoft/** — Microsoft-official eval content (Copilot Studio, Azure AI Eval SDK, Learn articles)
- **industry/** — Eval frameworks (Braintrust, LangSmith, Ragas, DeepEval), research papers, common patterns
- **concepts/** — Foundational evaluation concepts (grounding, faithfulness, test design, etc.)
- **changelog/** — Daily log of what was added or updated

## How content is organized

Each file covers a single topic in a layered format:
- **TL;DR** — one-liner summary
- **Key Facts** — bullet points of essentials
- **Customer Impact** — what anyone working with customers should know
- **Detail** — full writeup with quotes, examples, nuances
- **Sources** — links to original content

## How this repo is updated

An automated research agent runs daily, searching for new content across Microsoft docs, industry blogs, GitHub repos, and academic research. Findings are written as layered markdown files and committed with a changelog entry.
